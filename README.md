# Alpha Stack — Claude Code Skills for Crypto & DeFi

**Turn Claude Code into a crypto research terminal.**

Alpha Stack is a free, open-source skills pack for Claude Code. Drop in the markdown files and Claude Code gets structured, battle-tested research workflows for token DD, DeFi protocol evaluation, narrative tracking, on-chain signal reading, trade journaling, portfolio risk, and arbitrage spotting.

No plugins. No API keys. No setup beyond dropping in markdown files.

---

## What's Included

### 8 Skills

| Skill | What It Does |
|-------|-------------|
| `CLAUDE.md` | Sets crypto domain context — loads automatically |
| `token-research.md` | Full DD on any token: team, tokenomics, narrative fit, scored go/no-go |
| `defi-protocol-eval.md` | Safety and yield sustainability assessment for any DeFi protocol |
| `narrative-radar.md` | Map what narratives are hot, building, peaking, or fading — and find the best token for each |
| `onchain-intel.md` | Read whale moves and smart money signals before they hit social |
| `trade-journal.md` | Log trades and run pattern analysis to find what's actually costing you |
| `portfolio-risk.md` | Check positions against your risk limits before making a new trade |
| `arbitrage-scout.md` | Surface CEX/DEX spreads, funding rate anomalies, and stablecoin depegs — net of fees |

### 3 Workflows

| Workflow | What It Does |
|----------|-------------|
| `morning-alpha.md` | 10-minute morning ritual: market pulse, narratives, positions, today's best play |
| `token-dd-workflow.md` | End-to-end due diligence sequence from discovery to entry decision |
| `defi-entry-checklist.md` | Pre-entry safety checklist that surfaces the most common ways DeFi positions go wrong |

## Installation

1. Clone or download this repo
2. Copy the `skills/` folder into your Claude Code project root
3. Copy `CLAUDE.md` into your project root (or append to your existing `CLAUDE.md`)
4. Start a Claude Code session — your skills are active

## Why This Works

Claude Code is a powerful research tool but it needs domain context to be useful for crypto. Without it, you get generic answers that miss nuances like tokenomics red flags, DeFi yield sustainability, or what "smart money" on-chain signals actually mean.

Alpha Stack gives Claude Code that context — built by someone running active crypto scans and DeFi research every day.
