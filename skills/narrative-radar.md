# Skill: Narrative Radar

**Trigger:** "What narratives are hot?" / "Scan narratives" / "What's the market playing right now?"

## What This Does

Maps the current crypto narrative landscape — what's early, what's peaking, what's dying. Helps you find where smart money is rotating before it's crowded.

---

## Workflow

### Step 1 — Scan Active Narratives

Check across: Crypto Twitter (X), Reddit (r/CryptoCurrency, r/defi), CoinGecko trending, CoinMarketCap sector performance, major crypto news outlets.

For each active narrative, identify:
- Name and core thesis
- Primary tokens attached to it
- Estimated stage (see Stage Framework below)
- Corroborating signals (social, on-chain, VC activity, media)

### Step 2 — Stage Framework

| Stage | Characteristics | Risk/Reward |
|-------|----------------|-------------|
| **Emerging** | <3 sources, no mainstream coverage, price not yet moved | Highest risk, highest reward |
| **Building** | 3-10 sources, some CT attention, early price movement | Good entry window |
| **Peaking** | Mainstream crypto media covering it, significant price move already in | Momentum plays only — tight stops |
| **Saturated** | Everyone knows it, articles on Forbes/CNBC, late money coming in | Avoid new entries |
| **Fading** | Narrative is yesterday's news, rotation happening out | Short opportunity or exit |

### Step 3 — Score Each Narrative

| Criterion | Score (1-5) |
|-----------|-------------|
| Stage (5=Emerging, 1=Fading) | |
| Breadth of signal sources | |
| Token liquidity (can you enter/exit?) | |
| Catalyst visibility (is there a clear event?) | |
| Competition for the trade | |
| **Total** | **/25** |

### Step 4 — Token Shortlist

For each narrative scoring ≥15/25, list the 2-3 highest-signal tokens attached to it:
- Token name and ticker
- Why it's the best expression of this narrative
- Current market cap (smaller = more upside, but more risk)

### Step 5 — Output

```
## Narrative Radar — [DATE]

### [Narrative Name] — Stage: [X] — Score: X/25
**Thesis:** [One sentence]
**Signals:** [What corroborates this — be specific]
**Best tokens:** $TICKER1 (MC: $Xm), $TICKER2 (MC: $Xm)
**Entry window:** [Now / Wait for X / Already closed]
**Risk:** [Primary risk to this narrative]

---
[repeat for each narrative ≥15/25]

### Fading / Avoid
[Narratives to rotate out of]

### Nothing Actionable
[If no narratives meet threshold]
```
